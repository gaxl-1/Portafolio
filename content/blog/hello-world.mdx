# Hola Mundo

Bienvenido a mi nuevo portafolio. He construido este sitio usando **Next.js 15**, **Tailwind CSS** y **MDX**.

## ¿Por qué este stack?

1. **Rendimiento**: Next.js ofrece una velocidad increíble.
2. **Flexibilidad**: MDX me permite escribir contenido y usar componentes de React.
3. **Diseño**: Tailwind hace que estilar sea muy rápido.
